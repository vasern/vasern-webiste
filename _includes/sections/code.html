Setup database for your apps in seconds

  1. {% highlight ssh %} $ npm install vasern --save {% endhighlight %}
  2. {% highlight js lineos %} import Vasern, { types } from 'vasern'; var VasernDB = new Vasern({ schemas: [{ name: 'TodoItems', indexes: { createdOn: types.datetime(), name: types.string(55), status: types.boolean(), author: types.ref('Users') } }, { name: 'Users', indexes: { firstName: types.string(55), lastName: types.string(55), yearOfBirth: types.number() } }] }) {% endhighlight %}
  3. {% highlight javascript %} VasernDB.collect(“TodoItems”).insert({ createdOn: new Date(), name: “Be awesome!”, description: “Not too awesome” }); {% endhighlight %}
