Overview

Vasern is a data storage for React Native, focus on performance, and consistency. Our goal is to develop an open source, developer friendly, database sync solution.

This documentation will walk you through Vasern installation, work with schema, basic operations, and provide with examples.

Please let us know if you found a typo, incorrect or unclear information in this document.

Note: Vasern alpha is now available for experimental, subscribe to Vasern on Github for any update
