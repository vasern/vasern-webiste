## Data types

Data types used to define `Vasern` data schema.

```js
import { types } from 'vasern';
```

---

### primary

```js
types.primary(55)
```

---

### string

```js
types.string(55)
```

---

### boolean

```js
types.boolean()
```

---

### number

```js
types.number()
```

---

### ref

```js
types.ref('Users')
```
